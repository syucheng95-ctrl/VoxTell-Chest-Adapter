# 问题定位

本目录专门用于存放“为什么当前实验没有达到目标”的定位工作。

和其他目录的分工：

- `实验记录/`：记录已经完成的实验结果与阶段结论
- `实验计划/`：记录下一阶段结构设计与路线图
- `docs/plans/`：记录实现清单与执行计划
- `问题定位/`：记录**当前失败点/瓶颈的定位实验**、分析文档，以及为定位问题而写的辅助脚本

## 建议用法

当出现下面这类问题时，优先在本目录补材料：

- 指标为什么没有提升
- 某个模块是否真的起作用
- 训练统计改善了，但最终验证结果没有改善
- 不确定问题出在结构、监督、采样还是数据
