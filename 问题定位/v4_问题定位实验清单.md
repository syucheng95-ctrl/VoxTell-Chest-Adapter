# v4 问题定位实验清单

本文档只回答一件事：

**为什么 v4 的训练统计已经变好，但 `micro_dice` 没有改善，反而更低。**

当前已知现象：

- v4 训练统计：
  - `zero_loss_ratio = 0.0`
  - `suppression_mean_pos_avg = 0.530`
  - `suppression_mean_neg_avg = 0.478`
- v4 验证结果：
  - `mean_dice = 0.2202`
  - `micro_dice = 0.4062`
- 结论：
  - 训练行为更健康了
  - 但最终体素级误报没有被有效压住

因此，定位目标是判断：

1. suppression 是否真的在参与最终预测
2. 它是在帮忙还是在捣乱
3. 问题更偏向监督不对，还是结构粒度不对

---

## 实验 1：推理侧 suppression 消融

### 目的

先回答最关键的问题：

- `v4 full` 和 `v4 without suppression` 是否真的不同

### 要做什么

做 3 个推理版本：

1. `v4_full`
2. `v4_without_suppression`
   做法：推理时把 suppression 固定为全 1
3. `v4_without_category`
   做法：去掉 category embedding 的作用，只保留 group-wise suppression

### 要看什么结论

- 如果 `v4_full` 和 `without_suppression` 几乎一样
  - 说明 suppression 对最终 mask 的贡献极弱

- 如果 `v4_full` 比 `without_suppression` 更差
  - 说明 suppression 正在错误地起作用

- 如果 `v4_full` 只在某些类上更好
  - 说明 suppression 可能只对部分病灶模式有效

---

## 实验 2：按类别比较 baseline / v2 / v3 / v4

### 目的

判断 v4 的问题主要集中在哪些病灶类别。

### 要做什么

对以下版本做按类别比较：

- baseline
- v2
- v3
- v4

重点查看：

- `mean_dice`
- `micro_dice`
- `mean_precision`
- `mean_recall`

重点类别建议优先看：

- `ground_glass_opacity`
- `consolidation`
- `nodule`
- `pleural_effusion_or_thickening`
- `emphysema`

### 要看什么结论

- 如果 v4 主要在弥漫性病灶上退化
  - 说明 suppression 可能过度压制了大面积合理响应

- 如果 v4 主要在小目标类上没有改善
  - 说明 group-wise suppression 对小目标误报帮助有限

- 如果只有个别类提升
  - 说明 suppression 的作用可能是类别依赖的

---

## 实验 3：suppression 与误差的相关性分析

### 目的

判断 suppression 学到的信号，是否真的和最终错误体素对齐。

### 要做什么

在 patch 级别统计：

- `suppression_mean`
- `fp_penalty`
- `pred_voxels`
- patch-level precision
- patch-level recall

重点分别看：

- `positive`
- `hard_negative`
- `random_negative`

### 要看什么结论

- 如果 suppression 更低时，FP 没有明显变少
  - 说明 suppression 学到的是弱相关信号

- 如果 suppression 更低时，TP 先掉很多，而 FP 没掉多少
  - 说明 suppression 在“整体保守化”，不是精准抑制

- 如果 hard negative 上 suppression 和 FP 有明显关系
  - 说明监督有用，但还需要更细设计

---

## 实验 4：hard negative vs random negative 分开统计

### 目的

判断模型是否真的把 hard negative 当成更难的背景来处理。

### 要做什么

分别统计：

- `hard_negative` 的 suppression 均值
- `random_negative` 的 suppression 均值
- 两者平均 `pred_voxels`
- 两者平均 `fp_penalty`

### 要看什么结论

- 如果两者差不多
  - 说明当前 supervision 仍然太粗

- 如果 hard negative 的 suppression 明显更低，但 FP 仍然高
  - 说明 suppression 的作用粒度不够

---

## 实验 5：case review 可视化

### 目的

直接判断 suppression 的实际行为模式。

### 要做什么

挑 6 到 10 个 case：

- 2 个 v4 比 v3 好
- 2 个 v4 和 v3 差不多
- 2 个 v4 比 v3 差

对比：

- gt
- v3 prediction
- v4 prediction

重点问：

1. v4 是不是把 mask 整体缩小了
2. v4 压掉的是错误区域，还是连正确区域一起压掉
3. 额外误报集中在哪些结构附近

### 要看什么结论

- 如果常见模式是“正确和错误一起缩”
  - 说明 suppression 仍是整体保守化

- 如果错误主要集中在局部边缘/胸膜旁/血管旁
  - 说明问题开始指向空间粒度不足

---

## 实验 6：小规模训练消融

### 目的

确认问题来自：

- supervision 本身
- easy negative 过滤
- 还是 suppression 接法

### 要做什么

做 3 个小规模训练版本：

1. `seg loss only`
2. `seg + suppression loss`
3. `seg + suppression loss + skip easy negatives`

建议：

- 用 smoke 或 10-case pilot
- 不需要 fullrun

### 要看什么结论

- 如果 2 比 1 训练统计更好，但预测没更好
  - 说明 suppression loss 没有对准最终任务

- 如果 3 比 2 才更好
  - 说明 easy negative 处理是关键因素

---

## 建议执行顺序

建议先做：

1. 推理侧 suppression 消融
2. v2/v3/v4 按类别对比
3. suppression 与误差相关性分析
4. case review 可视化

这四步做完之后，再决定：

- 是先改 supervision
- 还是先改结构

---

## 当前阶段最可能的判断标准

如果前四步做完后发现：

- suppression 对结果影响很弱
  - 优先怀疑接法不对

- suppression 会明显改变结果，但主要表现为整体缩放
  - 优先怀疑监督太粗

- suppression 对 hard negative 有作用，但对最终 FP 没帮助
  - 优先怀疑粒度不够，需要更贴近空间错误

---

## 一句话目标

这套定位实验不是为了再跑一个更高的分，
而是为了明确回答：

**v4 的问题到底出在 suppression 没有效果，还是它有效果但作用错了地方。**
